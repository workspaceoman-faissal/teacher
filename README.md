---

## 🚀 رفع الموقع على GitHub Pages

### ١. إنشاء Repository
- اذهب إلى github.com وسجل دخولك
- اضغط **New repository**
- اسم الـ repo مثلاً: `teacher-eval`
- اختر **Public** ← مهم لتفعيل GitHub Pages مجاناً
- اضغط **Create repository**

### ٢. رفع الملفات
- في صفحة الـ repo اضغط **uploading an existing file**
- ارفع: `index.html` و `admin.html`
- أنشئ مجلد `data` وارفع فيه `feedbacks.json`
- اضغط **Commit changes**

### ٣. تفعيل GitHub Pages
- اذهب إلى **Settings** → **Pages**
- في Source: اختر **Deploy from a branch**
- Branch: **main** / Folder: **/ (root)**
- اضغط **Save**
- بعد دقيقة رابطك: `https://USERNAME.github.io/teacher-eval/`

---

## 🔑 إعداد الحفظ التلقائي عبر GitHub API

### إنشاء Personal Access Token
1. GitHub → **Settings** → **Developer settings**
2. **Personal access tokens** → **Tokens (classic)**
3. اضغط **Generate new token (classic)**
4. اكتب اسماً: `Teacher Eval Token`
5. فعّل: ✅ **repo** (كاملة)
6. اضغط **Generate token**
7. **انسخ الـ Token فوراً!**

### ربطه بالموقع
1. افتح `admin.html` على موقعك
2. سجّل دخول: user=`teacher` / pass=`1234`
3. اذهب لـ **مشاركة الرابط**
4. أدخل: اسم المستخدم، اسم الـ repo، الـ Token
5. اضغط **حفظ إعدادات GitHub**

كل رد جديد سيُحفظ تلقائياً في `data/feedbacks.json` ✅
