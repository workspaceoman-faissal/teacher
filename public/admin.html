🔑 متغيرات البيئة على Vercel

هذه المتغيرات تُضاف مرة واحدة في لوحة Vercel ولا تظهر في أي ملف. اذهب إلى: Vercel Dashboard → Project → Settings → Environment Variables 

# اسم مستخدمك على GitHub
GH_OWNER = your-github-username

# اسم الـ repository
GH_REPO  = teacher-eval

# Personal Access Token (صلاحية repo)
GH_TOKEN = ghp_xxxxxxxxxxxxxxxxxxxx

# كلمة مرور لوحة التحكم
ADMIN_PASS = your-strong-password

✅ بعد إضافتها اضغط Redeploy في Vercel
✅ لا يوجد أي بيانات سرية في ملفات الموقع
✅ حتى لو شاف أحد كود الموقع لن يجد أي token
